# Engenharia de Software com IA Aplicada

### Modulo 01 - Fundamentos de IA e LLMs para Programadores 
#### Professor: Erick Wendel,  
Aluno: Robson Fagundes

Ao longo das aulas, explorei os principais fundamentos por trás de sistemas de IA modernos, especialmente aqueles que utilizam o conceito de RAG — Retrieval-Augmented Generation. Agora, chegou o momento de transformar teoria em prática.

Meu primeiro projeto funcional de RAG utilizando apenas JavaScript e Neo4j.  
Minha intenção foi demonstrar que é possível construir uma arquitetura robusta de recuperação e consulta de informações sem depender de soluções prontas ou serviços de terceiros.
